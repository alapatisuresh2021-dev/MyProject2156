# Python LLM Projects Submodules

This branch contains git submodules for popular Python LLM projects.

## How to Download and Initialize the Projects

1. Clone the repository:  
   `git clone https://github.com/alapatisuresh2021-dev/MyProject2156.git`

2. Switch to the llm-branch:  
   `git checkout llm-branch`

3. Initialize and update submodules:  
   `git submodule update --init --recursive`

This will download all the submodule repositories into their respective directories.
